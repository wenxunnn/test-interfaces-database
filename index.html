About this project

A quick look at what the Interfaces Database is for.

This prototype visualizes a database of low-dimensional overlayers supported on metal substrates — the kind of surface structures studied in catalysis and electrochemistry research.

Use the Interface Explorer to pick elements for an overlayer and a substrate, then search for calculated systems that match. Click into any result to see its summary and crystal structure.

Right now this runs on a small set of placeholder systems so the search and browsing flow can be tested before real calculation data is connected.
