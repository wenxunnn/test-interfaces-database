About this project

A quick look at what the Interfaces Database is for.

This prototype visualizes a database of low-dimensional overlayers supported on metal substrates — the kind of surface structures studied in catalysis and electrochemistry research.

Use the Interface Explorer to pick elements for an overlayer and a substrate, then search for calculated systems that match. Click into any result to see its summary, POSCAR-derived structure card, and an interactive structure preview.

This version loads real calculation rows from interfaces_database.json, so the matching table, filters, detail pages, bulk references, and graph use one JSON-backed data source.
