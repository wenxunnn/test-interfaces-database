About this project

A quick look at what the Interfaces Database is for.

This prototype visualizes a database of low-dimensional overlayers supported on metal substrates — the kind of surface structures studied in catalysis and electrochemistry research.

Use the Interface Explorer to pick elements for an overlayer and a substrate, then search for calculated systems that match. Click into any result to see its summary, POSCAR-derived structure card, and initial input files.

This version loads real calculation rows from dft-interface-database-export.csv, so the matching table, filters, and detail pages use your exported interface database data.
